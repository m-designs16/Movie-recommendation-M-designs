# Movie-recommendation-M-designs


A Movie Recommendation System built using Python and Machine Learning that suggests similar movies based on content similarity.
It uses Cosine Similarity, TMDB API for posters, and a Streamlit web interface for user interaction.
Technologies: Python, Pandas, Scikit-learn, Streamlit, TMDB API, Jupyter Notebook, GitHub , Picharm
